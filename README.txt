FontCleanerGUI
==============

Что внутри:
- FontCleanerGUI.cs   -> исходник GUI-программы на C# WinForms
- build_exe.bat       -> сборка в .exe на Windows

Как собрать:
1. Скопируй папку на Windows.
2. Запусти build_exe.bat.
3. Если csc.exe не найден, установи Visual Studio Build Tools или .NET Framework Developer Pack.
4. После сборки появится FontCleanerGUI.exe.

Как использовать:
1. Запускай от имени администратора.
2. Сначала оставь галочку "Без удаления (предпросмотр)".
3. Нажми "Очистить шрифты" и проверь лог.
4. Потом сними галочку и запусти реальное удаление.

Важно:
- Утилита старается сохранить системные шрифты по whitelist.
- На некоторых сборках Windows список системных шрифтов может отличаться.
- После удаления рекомендуется перезагрузка.
